# Repository Relations

This section organizes the structural relations between repositories within the AI Text Project ecosystem.

Rather than treating repositories as isolated storage spaces,
this layer describes how different repositories function together as connected semantic environments.

The purpose of this layer is to clarify:

- repository roles
- layer relationships
- projection directions
- navigation continuity
- semantic coordination across repositories

## Projection Structure

The repository ecosystem is organized through layered projection pathways.

General flow:

Research Layer
↓
Bridge Layer
↓
Public Projection Layer

This structure allows:

- different density levels
- multiple reader entry points
- phase-adaptive navigation
- structure-preserving externalization

## Working Perspective

This layer treats repositories not only as technical containers,
